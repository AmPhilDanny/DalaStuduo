# SkillBridge Africa — Submission Info

## Theme
Education & Skills

## Description
SkillBridge Africa is a talent platform connecting African tech builders with opportunities. Students create rich profiles with portfolios and project history, discover collaborators through the talent directory, post and join projects, and apply for internships and part-time jobs posted by verified firms. AI-powered tools help polish profiles, draft applications, analyze skill gaps, and provide tutoring.
